# TextExtractFromPSD

## require

Require psd-tools package

```
pip install psd-tools
```

## how to use

Execute command line

```
python3 TextExtractFromPSD.py
```

and Specify target file. (PSD or PSB)

```
Target PSD file path: /Users/username/Downloads/sample.psd
```
